# O que é isso?
Essa é uma pequena pagina web que pega 3 tipos de dados do usuário: O nome de usuário, o email e uma senha. Ele possui alguns tipos de validação dos dados:
1. Ele verifica se os campos estão em branco
2. Verifica se o nome de usuario é muito pequeno
3. Se a senha é muito pequena ou muito grande
4. Se o nome de usuário começa com numeros
E impede que o usuário prossiga. E quando todos os dados estão corretos, ele apaga todos os campos e avisa ao usuário.
# A identidade visual
A paleta de cores foi inspirada no site https://pierre.finance/, que tem como destaque um verde neon, preto e branco.
